    What to choose¶

    • Cloud: Phaxio or Sinch Fax API v3
    • Self‑hosted: SIP/Asterisk (AMI + T.38)

    Checklist¶

    • Backend selection and credentials
    • Public URL and TLS (cloud backends)
    • API key and scopes
    • Storage and retention (inbound)
