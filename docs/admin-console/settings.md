# Settings (Admin Console)

Configure backend, security, storage/inbound, and MCP. Every field includes helper text and deep links.

- Backend: choose the active outbound provider; swapping providers is a guided flow.
- Security: require `X-API-Key`, enforce HTTPS, audit logging, rate limits.
- Storage & Inbound: local vs S3/S3-compatible; limits for inbound listing/download.
- MCP: enable Node/Python transports (stdio/HTTP/SSE), set OAuth for SSE.
