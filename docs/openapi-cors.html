🛠️ Tool-Specific Instructions

Postman

  1. Open Postman
  2. Click "Import" → "Link"
  3. Paste: https://raw.githubusercontent.com/DMontgomery40/Faxbot/docs-jekyll-site/docs/openapi.yaml
  4. Click "Continue" → "Import"

Insomnia

  1. Create new Request Collection
  2. Click "Import/Export" → "Import Data" → "From URL"
  3. Paste the raw GitHub URL

VS Code (OpenAPI Extension)

  1. Install "OpenAPI (Swagger) Editor" extension
  2. Open Command Palette (Cmd/Ctrl+Shift+P)
  3. Run "OpenAPI: Show Preview"
  4. Use the raw GitHub URL

🔧 Why This Works

GitHub's raw content delivery (raw.githubusercontent.com) serves files with proper CORS headers, allowing external tools to access the OpenAPI specification without cross-origin restrictions.
