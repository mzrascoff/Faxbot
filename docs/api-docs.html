🚀 Faxbot API Documentation

The first and only open-source, self-hostable fax API.

📋 Copy & Paste URLs (100% Working)

Use these URLs in any API documentation tool:

JSON Format (Recommended)

https://docs.faxbot.net/openapi.json
🔍 Open in Swagger UI

YAML Format

https://docs.faxbot.net/openapi.yaml
🔍 Open YAML in Swagger UI

🛠️ Tool-Specific Instructions

Postman

  1. Click "Import" → "Link"
  2. Paste: https://docs.faxbot.net/openapi.json
  3. Click "Continue" → "Import"

Insomnia

  1. File → Import/Export → Import Data → From URL
  2. Paste: https://docs.faxbot.net/openapi.json

Stoplight Studio

  1. Create New Project → Import OpenAPI
  2. Paste: https://docs.faxbot.net/openapi.json

Redoc

  1. Go to redocly.github.io/redoc
  2. Add ?url=https://docs.faxbot.net/openapi.json to the URL

⚡ Quick API Test

Test the API directly:

Health Check

curl http://localhost:8080/health

Send a Fax

curl -X POST "http://localhost:8080/fax" \
  -H "X-API-Key: your_key_here" \
  -F "to=+15551234567" \
  -F "file=@document.pdf"

Check Status

curl -H "X-API-Key: your_key_here" \
  "http://localhost:8080/fax/{job_id}"

📚 Core Endpoints

Method Endpoint Description
POST /fax Send a fax
GET /fax/{id} Get fax status
GET /health Health check
GET /inbound List received faxes
